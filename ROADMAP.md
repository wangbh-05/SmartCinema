## 变更摘要 (2026-07-07 v2)

### 新增文件
| 文件 | 说明 |
|------|------|
| `order.html` | 独立订单确认页面（座位明细/费用/确认支付/取消返回） |

### 修改文件
| 文件 | 变更类型 | 说明 |
|------|---------|------|
| `src/core/Cinema.js` | v5→v6 | 白底主题、热度边框替代光晕、空座白色、已售灰色、选座独立切换、银幕曲线光锥 |
| `src/core/SeatData.js` | 新增方法 | `getAllOccupiedKeys()` 供订单页使用 |
| `src/modules/ScoreEngine.js` | 修复 | 评分计算*10修复、权重调整为真实影院体系(35/30/20/15) |
| `src/app.js` | 重构 | 级联推荐表单、订单跳转、历史订单折叠、移除HeatmapEngine依赖 |
| `index.html` | 重构 | 推荐表单级联重构、订单区改为提交按钮、删除独立热度图、删除canvas-info |
| `REQUIREMENTS_CHECK.md` | 更新 | 同步v2所有变更、统计表更新至~85/110 |
| `ROADMAP.md` | 更新 | 进度更新至70%、变更摘要补充 |

### 移除
| 文件 | 说明 |
|------|------|
| `src/modules/HeatmapEngine.js` | 独立热度图不再使用（热度整合到座位边框） |

### 关键架构变更
1. **热度显示**：独立Canvas + 荧光光晕 → 座位边框颜色（常驻、柔和）
2. **主题**：深色 → 白底浅色
3. **选座**：Ctrl多选 → 独立切换
4. **订单**：侧边栏直接创建 → 独立确认页 order.html
5. **推荐**：固定表单 → 级联拓扑（人数→年龄段→类型动态关联）

## 优先级排序（更新后）

1. **P1 - 模块完善**：热度一周切换、观众评分、色盲模式、表单完善
2. **P2 - 设计文档**：视觉设计文档、用户流程图、AI说明书、report.pdf
3. **P3 - 加分项**：AI顾问、动画优化、WebSocket
