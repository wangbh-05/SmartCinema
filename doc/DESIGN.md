## 一、设计理念

SmartCinema 定位为**科技感影院选座系统**，整体视觉传达"专业、智能、沉浸"的品牌感受。设计遵循以下原则：

- **暗色沉浸**：深色背景模拟影院环境，减少视觉疲劳
- **Canvas 白底**：座位图使用白底以确保座位色彩对比清晰、热度边框可见
- **卡片分层**：右侧面板使用圆角卡片+微妙边框，信息层次分明
- **低认知负荷**：≤3步完成选座，关键操作（推荐/提交）使用醒目按钮色

## 三、布局设计

### 3.1 整体布局（Grid）

```
┌──────────────────────────────────────────────┐
│  Header (sticky, 12px 24px padding)          │
│  Logo | 厅选择 | 日期 | 用户 | 按钮组          │
├────────────────────────────┬─────────────────┤
│                            │  🤖 智能推荐     │
│                            │  ①人数→②年龄段   │
│   🪑 选座                  │  →③类型→④姓名    │
│   [Canvas 座位图]          │  [执行推荐]      │
│   工具栏: 推荐|手动|清空    │                 │
│                            ├─────────────────┤
│   统计: 空座|已选|已售      │  📊 体验评分    │
│                            │  总分+四维度     │
│                            │  手动评分面板    │
│                            ├─────────────────┤
│                            │  🛒 提交订单     │
│                            │  摘要+按钮       │
│                            │  历史订单        │
├────────────────────────────┴─────────────────┤
│  ⚙️ 设置（全宽）                             │
└──────────────────────────────────────────────┘
```

- **Grid**：`1fr 360px`，Canvas 占据左侧剩余空间，右侧固定 360px 侧边栏
- **响应式**：≤900px 时切换为单列布局
- **最大宽度**：1300px 居中

### 3.2 Canvas 座位图布局

- **弧形排列**：圆形弧线算法，每排独立半径 `R = baseR + row * pitch * 0.55`
- **行列标签**：行号标注在左侧（"1排"～"10排"），列号标注在上方
- **座位间距**：小厅 38px / 中厅 30px / 大厅 22px（自适应）
- **过道**：>=14列时自动插入2列过道，>=8列插入1列
- **银幕**：顶部居中，二次贝塞尔曲线，光锥向下照射

### 3.3 组件间距规范

| 元素 | 值 |
|------|-----|
| 卡片圆角 | 10px（外层）/ 8px（内部） |
| 卡片内边距 | 20px |
| 卡片间距 | 20px |
| 按钮圆角 | 6px（标准）/ 10px（大按钮） |
| 表单元素圆角 | 6px |

## 五、风格说明

### 5.1 科技感表达

- **暗色基调**：GitHub-dark 风格的深色背景
- **蓝色强调**：`#58A6FF` 用作标题色和交互反馈
- **金色点缀**：Logo 和价格使用 `#FDD835` 提亮
- **微妙边框**：`#30363D` 的卡片边框在深背景下低调但不失层次

### 5.2 交互反馈

| 交互 | 反馈 |
|------|------|
| 按钮悬停 | 边框变蓝 `#58A6FF`，背景变亮 `#1F2A38` |
| 卡片悬停 | 边框变亮 `#3B4A5E` |
| 座位悬停 | 径向渐变光晕（选中=琥珀/推荐=紫/空座=浅灰） |
| 拖拽框选 | 虚线矩形 `#FBBF24` 黄色 |
| 选中座位 | ✓ 图标，填充由白变琥珀 |

### 5.3 无障碍考量

- **焦点可见**：`:focus-visible` 2px 蓝色轮廓
- **语音提示**：Web Speech API，切换厅/日期/操作均播报
- **色盲模式**：蓝/橙配色替代红/绿，Canvas + CSS 同步切换
- **大字体模式**：`body.accessibility-mode` 放大至 18px，按钮最小 48px
- **减少动画**：`prefers-reduced-motion` 媒体查询

### 5.4 Canvas 白底主题决策

座位图选择白底而非跟随页面深色主题，原因：

1. **热度可见性**：有色边框（红/琥珀/蓝）在白色背景上对比度远优于深色背景
2. **真实影厅感**：纸质选座图/票根通常是白底
3. **色彩准确性**：已售灰色、空座白色在深底上难以区分
4. **打印友好**：白底 Canvas 可直接用于截图/打印

## 七、响应式断点

| 断点 | 布局变化 |
|------|---------|
| >900px | 双列 Grid（Canvas + 侧边栏） |
| ≤900px | 单列堆叠，侧边栏移至 Canvas 下方 |
| Canvas | `maxW = min(viewport-32px, 1100px)`，自适应缩放 |

## 八、动画与过渡

| 场景 | 动画 |
|------|------|
| 按钮/卡片悬停 | `transition: all 0.2s` |
| 认证模态框 | `display: flex` 切换 |
| Toast 通知 | `fadeInOut 2s ease`（order.html） |
| 推荐结果 | `display: block` 展开 |

> 注：Canvas 内部无动画，redraw() 为全量重绘以保证性能稳定。
