# SmartCinema

SmartCinema 是一个无运行时依赖的原生 JavaScript 影院购票演示。消费者入口为 `index.html`，覆盖“选择场次 → 选择票种与数量 → 选择座位 → 锁座 → 登录/注册 → 确认订单”的完整本地闭环。

## 启动

需要 Node.js 18 或更高版本。

```bash
npm install
npm start
```

在浏览器打开 <http://localhost:8080>。

## 功能

- 7 部经典影片、3 家虚构海淀影院与从本地当天起的 3 日滚动排期；
- 影片和影院均支持横向滑动浏览，影片海报使用本地资源；
- 成人、儿童、学生、银发等票种，1–20 张票与座位数量严格一致；
- 情侣、亲子、长者、多人同行等连座推荐与座位体验提示；
- 100 / 200 / 300 座影厅、无障碍座位与移动端适配；
- 本地锁座、登录注册、订单确认、订单查询与退票演示；
- 大字体、高对比、色觉友好、减少动态等辅助功能。

影片为真实条目，封面素材已随项目保存于本地。影院名称仅借用北京海淀地名；地址、影厅、排期、价格和库存均为虚构演示数据，不代表真实影院信息。

## 项目结构

```text
SmartCinema/
├── index.html                  # 消费者入口
├── public/                     # 样式与本地海报资源
├── src/
│   ├── domain/                 # 业务规则与实体
│   ├── application/            # 购票、账户、推荐与偏好用例
│   ├── infrastructure/         # 演示目录、浏览器与本地存储
│   ├── ui/                     # DOM 控制器与组件
│   ├── commercial.js           # 页面编排
│   └── bootstrapCommercial.js  # 应用组合根
└── scripts/server.js           # 本地静态服务
```

## 数据说明

状态保存在当前浏览器的 LocalStorage 中，仅用于本地演示。请勿输入真实密码或个人信息；本项目不提供真实支付、验票或线上退款服务。
