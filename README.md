# SmartCinema

SmartCinema 是一个无运行时依赖的原生 JavaScript 影院购票演示。消费者入口为 `index.html`，覆盖“选择场次 → 选择票种与数量 → 选择座位 → 锁座 → 登录/注册 → 确认订单”的本地交易闭环。

## 使用方式

直接用浏览器打开 `index.html`，或通过任意静态文件服务器访问项目根目录。

## 功能

- 7 部经典影片，以及五道口、清河、中关村 3 家虚构演示影院；
- 影片和影院支持横向浏览，影片海报使用本地资源；
- 成人、儿童、学生、银发等票种，单笔 1–20 张，且票数与座位数严格一致；
- 情侣、朋友、亲子、银发、团体等同行方式的连座推荐与座位体验提示；
- 小、中、大三种影厅，分别提供 100、200、300 个弧形 Canvas 座位；
- 座位热度支持按周一至周日及一周综合查看；
- 本地锁座、登录注册、订单确认、订单查询与退票演示；
- 大字体、高对比、色觉友好、减少动态等辅助功能。

## 项目结构

```text
SmartCinema/
├── index.html                  # 消费者入口
├── public/                     # 样式与本地海报资源
├── scripts/server.js           # 本地静态服务
└── src/
    ├── domain/                 # 业务规则与实体
    ├── application/            # 购票、账户、推荐与偏好用例
    ├── infrastructure/         # 演示目录、浏览器与本地存储
    ├── ui/                     # Canvas 视图、控制器与组件
    ├── commercial.js           # 页面编排
    └── bootstrapCommercial.js  # 应用组合根
```

领域和应用层不直接访问 DOM、LocalStorage、系统时间或随机源；浏览器能力由基础设施层通过组合根注入。

## 数据说明

状态仅保存在当前浏览器的 LocalStorage 中，用于本地演示。影院名称、道路地址、影厅、排期、价格和库存均为虚构数据，不代表真实影院信息。请勿输入真实密码或个人信息；本项目不提供真实支付、验票或线上退款服务。
